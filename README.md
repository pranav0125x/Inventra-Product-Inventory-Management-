# Inventra-Product-Inventory-Management-

📝 Project Overview
 • Inventra is a full-stack web application designed to track and manage product stock levels in real-time . It provides a centralized platform for admins to monitor inventory, perform CRUD operations, and avoid stock shortages or overstocking .

🚀 Core Features
 • Secure Admin Login: Restricts access to authorized users via credential verification .
 • Real-time Dashboard: Displays total product count, total inventory value, and low-stock alerts .
 • Complete Product Management: Full Create, Read, Update, and Delete (CRUD) capabilities for inventory records .
 • Smart Search & Filter: Instant product lookup by name and category-based sorting .
 • Data Validation: Prevents duplicate entries and ensures only valid data is stored .

🛠️ Technology Stack
 • Frontend: React.js (Vite) 
 • Backend: Java (Spring Boot) 
 • Database: MySQL 
 • Communication: REST API (Axios) 

⚙️ How It Works
 • The user interacts with the React frontend, which sends requests to the Spring Boot backend . The backend processes the logic, validates data, and communicates with the MySQL database through JPA to ensure all records are accurate and persistent .
