# [cite_start]Inventra - Product Inventory Management System [cite: 1]

### 📝 Project Overview
* [cite_start]**Inventra** is a full-stack web application designed to track and manage product stock levels in real-time[cite: 10, 11]. 
* [cite_start]It provides a centralized platform for admins to monitor inventory, perform CRUD operations, and avoid stock shortages[cite: 14, 21].

### 🚀 Core Features
* [cite_start]**Secure Admin Login**: Restricts access to authorized users via credential verification[cite: 150, 151].
* [cite_start]**Real-time Dashboard**: Displays total product count, total inventory value, and low-stock alerts[cite: 164, 165].
* [cite_start]**Complete Product Management**: Full Create, Read, Update, and Delete (CRUD) capabilities[cite: 38, 39, 153].
* [cite_start]**Smart Search & Filter**: Instant product lookup by name and category-based sorting[cite: 155, 157].
* [cite_start]**Data Validation**: Prevents duplicate entries and ensures only valid data is stored[cite: 160, 163].

* [cite_start]**Frontend**: React.js (Vite) [cite: 12]
* [cite_start]**Backend**: Java (Spring Boot) [cite: 12]
* [cite_start]**Database**: MySQL [cite: 133, 171]

* [cite_start]The user interacts with the **React** frontend, which sends requests to the **Spring Boot** backend[cite: 117, 120].
